# Newborn Valley Doula Care

Bilingual React/Vite website for a postpartum doula care business.

Live site:

```text
https://oglerma.github.io/newborn-valley-doula-care/
```

## Run Locally

```bash
npm install
npm run dev
```

Then open:

```text
http://127.0.0.1:5173/
```

The source `index.html` is not meant to be opened directly with `file://`.
